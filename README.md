# MyPoint

Реализованы веб-интерфейсы для кейса MyPoint.
-просмотр заказов;__
-фильтрация заказов;__
-удаление;__
-добавление заказа;__
-отображение координат курьеров и хабов на карте;

## Установка и запуск проекта
```
npm install
```
```
npm run serve
```

## Production build
```
npm run build
```
